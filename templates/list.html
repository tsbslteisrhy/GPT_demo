GPT-3란?

GPT-3는 OpenAI사에서 개발한 대형 언어 모델(Large Language Model)로 약 3,000억 개의 데이터 세트를 학습하고 1,750억 개의 파라미터를 가지고 있습니다. 이는 2020년 기준 가장 큰 모델 대비 약 10배 이상의 파라미터를 가지는 것입니다.
따라서 과거 특정 태스크(번역, 키워드 추출, 요약 등) 하나에 국한하여 우수한 성능을 보이던 모델들과는 다르게 GPT-3는 모든 자연어 태스크에서 우수한 성능을 보입니다.

[ 언어 모델 파라미터 수의 변화 ]

Amaranth10 + GPT-3

GPT-3는 특히 언어 생성에 우수한 모델로 Amaranth10에 GPT-3 기술을 접목한다면 아래와 같은 태스크에서 유의미한 성과를 보일 것으로 기대됩니다.

  • 번역
    GPT-3는 단어 간 치환 형식의 직역이 아닌 의역을 활용하여 자연스러운 번역이 가능합니다. 또한 다양한 언어를 한 번에 학습하여 제약없이 다양한 언어로 번역이 가능합니다.
  • 키워드 추출
    키워드 추출은 검색, 유사 문서 분류, 핵심 내용 파악 등에 활용되어 자연어 처리에서도 활발히 연구되고 있는 분야입니다. GPT-3의 Ada 모델은 속도와 비용면에서 키워드 추출 태스크에 추천되는 모델로 알려져있습니다.
  • 요약
    과거 언어 모델은 전체 문서에서 핵심 문장을 골라내는 요약 방식을 취했습니다. 그러나 GPT-3는 문맥을 이해하고 새로운 요약 문장을 생성하여 더 자연스럽고 창의적인 요약 결과가 도출됩니다.
  • 쪽지 자동 생성
    GPT-3는 언어 생성에 우수한 모델입니다. 특히나 몇 가지의 예시 데이터를 입력하면 짧은 시간에 스스로 학습하여 추론(Few-shot Learning)할 수 있습니다. 따라서 특정 형식에 맞는 새로운 문장을 자동 생성하는 등의 활용이 가능해집니다.
